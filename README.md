# til.webdonalds.org

> Visit [website](https://til.webdonalds.org)!

## Development

Developed with [Remix](https://remix.run). Required nodejs 16 or greater.

```shell
yarn
yarn dev
```

Open up [http://127.0.0.1:3000].

### Environment variables

- `AUTH0_DOMAIN`
- `AUTH0_CLIENT_ID`
- `AUTH0_CLIENT_SECRET`
- `AUTH0_CALLBACK_URL`
- `AUTH0_LOGOUT_URL`
- `HASURA_ADMIN_SECRET`

## Deployment

```shell
yarn build
yarn start
```
